# Angular

## Setup
**Install Angular**
`npm install -g @angular/cli@latest`

**Create Project**
`ng new name-of-app --no-strict`

**Install Bootstrap**
`npm install --save bootstrap@VERSON`

**Run server**
`ng serve`

## Useful Commands
**Add Component**
`ng generate component COMPONENT-NAME`
`ng g c COMPONENT-NAME`
